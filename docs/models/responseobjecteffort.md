# ResponseObjectEffort

Reasoning effort level (none/minimal for fast, low/medium for balanced, high/xhigh for thorough)


## Values

| Name      | Value     |
| --------- | --------- |
| `NONE`    | none      |
| `MINIMAL` | minimal   |
| `LOW`     | low       |
| `MEDIUM`  | medium    |
| `HIGH`    | high      |
| `XHIGH`   | xhigh     |
